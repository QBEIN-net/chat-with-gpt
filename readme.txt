== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/chat-with-gpt` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to plugin settings page and put your chatGPT apikey and configure plugin

== Frequently Asked Questions ==

= If AI do not answer on question =

Please check that you put correct and working apikey in apikey field in plugin settings page. Also check is your server IP country not in ban list (see description for check link)

= What happened when user used all requests by limit (per day) =

He got submit email form to get more answers from site owner

= Why I do not see chat on site page =

Check that you are logged in or checkbox *Show chat for guest users* on plugin settings page is unchecked (to show chat for guest users)
